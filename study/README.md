# Study Folder Guide

이 폴더는 날짜별 학습 기록을 쌓는 공간이다.

기본 규칙:

- 하루마다 `YYYY-MM-DD` 형식의 폴더를 만든다
- 그날 공부한 주제마다 `[주제].md` 파일을 하나씩 만든다
- 통과한 파일은 이름 끝에 `_finished.md`를 붙인다
- 파일명은 너무 길지 않게 쓴다
- 한 파일에는 한 주제만 정리한다
- 공부가 끝나면 체크리스트와 연결해서 복습한다

예시:

```text
study/
  2026-04-20/
    네트워크_finished.md
    운영체제.md
  2026-04-21/
    자료구조.md
```

추천 파일명:

- `cs.md`
- `network.md`
- `algorithm.md`
- `data-structure.md`
- `os.md`
- `database.md`
- `computer-architecture.md`
- `security.md`

원하는 경우 한글 파일명을 써도 된다. 중요한 것은 매일 같은 기준으로 누적하는 것이다.

작성 순서:

1. 오늘 공부할 주제를 정한다
2. `study/[날짜]/[주제].md` 파일을 만든다
3. 아래 템플릿 순서대로 작성한다
4. 끝나면 `checklist.md`에서 관련 항목을 체크한다
5. 통과하면 파일명을 `_finished.md`로 바꾼다
6. 마지막에 "내가 아직 헷갈리는 것"을 반드시 남긴다

기본 학습 흐름:

- 체크리스트의 미완료 항목 하나를 고른다
- 짧은 테스트를 먼저 푼다
- 부족한 부분만 보강한다
- 재테스트를 본다
- 통과 기준을 넘으면 체크하고 `_finished` 처리한다

세부 기준은 `study/GUIDE.md`의 `공통 통과 기준`, `기본 진행 방식`, `다음번 시작 규칙`을 따른다.
