# Study Checklist

이 파일은 CS 기초 학습용 체크리스트다. 각 항목은 `- [개념] [실습] 내용` 형식으로 관리한다.

- 첫 번째 칸: 개념을 이해하고 설명할 수 있으면 체크한다
- 두 번째 칸: 직접 실습하거나 적용해봤으면 체크한다
- 예시: 개념만 끝났으면 `- [x] [ ] 내용`, 개념과 실습을 모두 했으면 `- [x] [x] 내용`

## 공통 학습 루틴

- [ ] [ ] 각 주제마다 "한 줄 정의"를 직접 적었다
- [ ] [ ] 각 주제마다 "왜 중요한가"를 설명할 수 있다
- [ ] [ ] 각 주제마다 대표 예시를 1개 이상 적었다
- [ ] [ ] 각 주제마다 헷갈리는 개념과 비교 대상을 정리했다
- [ ] [ ] 각 주제마다 스스로 만든 확인 문제를 2개 이상 적었다
- [ ] [ ] 공부한 내용을 5줄 이하로 다시 요약했다
- [ ] [ ] 주 1회 전체 복습 시간을 따로 잡았다
