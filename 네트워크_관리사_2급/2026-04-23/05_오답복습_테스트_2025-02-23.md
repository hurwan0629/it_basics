# 오답 복습 + 테스트

대상 기출: `2025-02-23`

이번 오답은 공통적으로 `비슷한 명령어/용어를 헷갈린 경우`가 많다.
따라서 이번 복습은 `정답 하나 암기`보다 `비슷한 보기끼리 구분`하는 데 초점을 둔다.

---

## 1. 오답 복습

## 37번

### 문제 핵심
- Linux 서버를 `종료`하는 명령어가 `아닌 것`을 고르는 문제

### 정답
- `init 6`

### 왜 틀리기 쉬운가
- `shutdown`, `poweroff`, `halt`, `init` 계열이 전부 시스템 종료/재시작과 관련 있어서 헷갈리기 쉽다.

### 시험용 정리
- `shutdown -h now` : 즉시 종료
- `poweroff` : 전원 종료
- `halt` : 시스템 정지/종료
- `init 6` : 재부팅

### 한 줄 암기
- `init 6 = reboot`

---

## 38번

### 문제 핵심
- DNS 캐시가 갱신되지 않아 예전 도메인 정보로 접속될 때, 이를 초기화하는 명령어를 묻는 문제

### 정답
- `ipconfig /flushdns`

### 헷갈리는 보기 구분
- `ipconfig /displaydns` : 현재 DNS 캐시 보기
- `ipconfig /flushdns` : DNS 캐시 삭제
- `ipconfig /release` : DHCP IP 반납
- `ipconfig /renew` : DHCP IP 재할당 요청

### 한 줄 암기
- `flush = 비운다`

---

## 40번

### 문제 핵심
- Apache `httpd.conf`에서 POST 요청 등 클라이언트 요청 크기를 제한하는 지시자

### 정답
- `LimitRequestBody`

### 시험용 정리
- Apache에서 요청 바디 크기 제한은 `LimitRequestBody`
- 이름이 그럴듯해 보여도 `KeepRequestSize`, `RestrictBodyRequest`, `PostRequestSize`는 정답이 아니다.

### 한 줄 암기
- `요청 바디 제한 = LimitRequestBody`

---

## 43번

### 문제 핵심
- Linux에서 사용자에게 할당되어 사용자가 임의로 사용할 수 있는 디렉터리 영역

### 정답
- `Home Directory`

### 왜 틀리기 쉬운가
- `Temporary Directory`는 임시파일 저장 영역일 뿐, 특정 사용자 전용 공간이 아니다.

### 시험용 정리
- `Root Directory` : 최상위 디렉터리
- `Home Directory` : 각 사용자에게 할당된 기본 작업 공간
- `Temporary Directory` : 임시 파일 영역

### 한 줄 암기
- `내 공간 = Home Directory`

---

## 49번

### 문제 핵심
- 인가된 사용자와 장치만 내부망 접속을 허용하는 기술

### 정답
- `NAC (Network Access Control)`

### 헷갈리는 보기 구분
- `NAC` : 네트워크 접근 제어
- `NAT` : 사설/공인 IP 변환
- `WAF` : 웹 애플리케이션 공격 방어

### 시험용 정리
- 내부망 접속 전에 `누가 접속하는지`, `인가됐는지`, `정책 위반인지`를 검사하는 느낌이면 `NAC`

### 한 줄 암기
- `Access Control = NAC`

## 2. 한 번에 묶어 외우기

### 명령어
- `init 6` : 재부팅
- `ipconfig /flushdns` : DNS 캐시 초기화

### 설정
- `LimitRequestBody` : Apache 요청 바디 크기 제한

### Linux 디렉터리
- `Home Directory` : 사용자 개인 작업 공간

### 보안 용어
- `NAC` : 네트워크 접근 제어

## 3. 재테스트

답은 `번호 + 한 줄 근거`로 적는다.

1. 다음 중 Linux 시스템을 `재부팅`하는 명령어는?
   1. `halt`
   2. `shutdown -h now`
   3. `init 6`
   4. `poweroff`
   정답: 3

2. 다음 중 DNS 캐시를 초기화하는 명령어는?
   1. `ipconfig /displaydns`
   2. `ipconfig /flushdns`
   3. `ipconfig /release`
   4. `ipconfig /renew`
   정답: 2

3. Apache `httpd.conf`에서 클라이언트 요청 바디의 최대 크기를 제한하는 지시자는?
   1. `KeepRequestSize`
   2. `LimitRequestBody`
   3. `PostRequestSize`
   4. `RestrictBodyRequest`
   정답: 2

4. Linux에서 각 사용자에게 기본적으로 할당되는 개인 작업 디렉터리는?
   1. Root Directory
   2. Temporary Directory
   3. Home Directory
   4. Public Directory
   정답: 3
5. 인가된 사용자와 단말만 내부망에 접속하도록 제어하는 기술은?
   1. NAT
   2. NAC
   3. WAF
   4. DHCP
   정답: 1

6. 다음 중 `DHCP IP 주소 해제`와 관련된 명령어는?
   1. `ipconfig /displaydns`
   2. `ipconfig /flushdns`
   3. `ipconfig /release`
   4. `ipconfig /renew`
   정답: 1

7. 다음 설명에 해당하는 용어는?
   `웹 애플리케이션 공격을 방어하기 위한 보안 장비/기술`
   1. NAC
   2. NAT
   3. WAF
   4. ARP
   정답: 3

8. 다음 중 설명이 옳은 것은?
   1. `Temporary Directory`는 사용자 개인 전용 디렉터리이다.
   2. `Home Directory`는 사용자가 기본적으로 사용하는 개인 영역이다.
   3. `Root Directory`는 임시 파일 저장용 디렉터리이다.
   4. `Public Directory`는 모든 Linux에서 사용자 기본 디렉터리이다.

## 4. 재테스트 정답

1. `3`
   `init 6`은 재부팅이다.

2. `2`
   DNS 캐시 초기화는 `ipconfig /flushdns`다.

3. `2`
   Apache 요청 바디 제한 지시자는 `LimitRequestBody`다.

4. `3`
   사용자 개인 작업 영역은 `Home Directory`다.

5. `2`
   내부망 접근 통제는 `NAC`다.

6. `3`
   DHCP IP 주소 해제는 `ipconfig /release`다.

7. `3`
   웹 애플리케이션 방어는 `WAF`다.

8. `2`
   사용자 기본 작업 공간은 `Home Directory`다.
